# Session Summary — 7NORTHEAST Brand Color Work

## 1. Source Color Extraction

You uploaded an image of a minimalist black cat illustration on a deep red textured background and asked for the exact color code of the background.

**Method:** Sampled pixels across the background (avoiding the cat) and averaged a region to account for the fabric-texture variation in the image.

**Result:**
- **HEX:** `#8C2324`
- **RGB:** 140, 35, 37
- Described as a deep brick / oxblood red with slight tonal variation from the canvas-like texture.

This color became the foundation for the brand work that followed.

## 2. Brand Color System for "7NORTHEAST"

Using `#8C2324` (named **Oxblood**) as the primary brand color, a full color theory system was built out and delivered as a standalone HTML reference document (`7ne-color-system.html`).

### Core color math
- **HSL of base color:** 359°, 60%, 34%

### Palette components
| Role | Color | Hex |
|---|---|---|
| Primary | Oxblood | `#8C2324` |
| Tint (light) | Blush | `#EDB6B7` |
| Shade (dark) | Bordeaux | `#5A1618` |
| Shade (darkest) | Iron | `#291E1F` |
| Complementary accent | Verdigris (teal) | `#238C8A` |
| Analogous detail | Brass | `#B5893A` |
| Background | Linen | `#F5F1EA` |
| Background | Warm White | `#FAF8F4` |

### Color theory rationale
- **Complementary:** Verdigris sits at ~179° on the wheel (true opposite of 359°) — chosen deliberately because it also reads as oxidized copper roofing, tying the palette back to Northeast architecture rather than being an arbitrary teal.
- **Analogous:** Brass (~29°) chosen as the warm neighbor for fine detail (hardware/rivet feel); the other analogous neighbor (plum, ~329°) was noted but excluded from the working palette since two similar warm reds compete rather than complement.
- **Triadic:** Green (~119°) and blue (~239°) were calculated for reference but not adopted into the working system.
- **Tonal scale:** A 9-step tint/shade scale (50–900) was generated by holding hue and saturation constant and varying lightness only, so the brand red can extend into backgrounds, hover states, and dark mode without feeling like a different color.

### Usage ratio guidance
A recommended composition split for layouts:
- ~55% neutral field (Warm White / Linen)
- ~20% Iron (text/structure)
- ~18% Oxblood (brand moments)
- ~5% Verdigris (accent/interaction)
- ~2% Brass (fine detail)

### Accessibility (WCAG 2.1 contrast ratios)
| Pairing | Ratio | Result |
|---|---|---|
| Oxblood on Warm White | 8.8:1 | AAA |
| Warm White on Oxblood | 8.8:1 | AAA |
| Warm White on Iron | 16.2:1 | AAA |
| Oxblood on Iron | 1.8:1 | Fails — decorative only |
| Verdigris on Warm White | 4.1:1 | AA (large text/UI only) |

### Do / Don't guidance
- **Do** let Oxblood carry one hero moment per layout, surrounded by neutral space.
- **Don't** fill large background fields with full-strength Oxblood behind body copy — use the 700/900 shades instead.
- **Do** reserve Verdigris for interactive/informational elements only.
- **Don't** pair Oxblood with the plum analogous neighbor — too visually competitive.

**Deliverable:** `7ne-color-system.html` — a self-contained, styled reference document covering all of the above.
